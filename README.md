How to add a bug
- Press "Issues" button
- Press the "New Issue" button
- Fill in the issue mentioning the following items:
      - Device
      - Kernel version
      - Android version
      - The issue and steps to reproduce the issue
      - Logs if available
- Press "Submit new issue" button
